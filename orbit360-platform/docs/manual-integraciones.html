Orbit 360 · Manual de integraciones

Integraciones: configuración y valor

Cada integración se activa en Configuración → Integraciones (tarjeta con API key / webhook / OAuth + botón Probar) y, las de IA, en Automatizaciones → Motor de IA. Aquí: qué hace cada una, cómo se configura y qué valor aporta.

🔧 Cómo configurar (general): Configuración → Integraciones → elige la tarjeta → "⚙ Configurar" → pega API key / webhook / OAuth / endpoint → "🔌 Probar conexión" → marca "Integración activa" → Guardar. Las credenciales persisten por cliente (tenant). La conexión real se valida al activar el backend en migración.

📲 Comunicación y notificaciones

🔗 Make (orquestador)

Hub central de automatización: recibe los eventos de Orbit (póliza emitida, cobro vence, solicitud de gestión) y los ramifica a WhatsApp, correo, Sheets, etc.
Config: pega la URL del webhook del escenario Make.
Valor: automatiza todo el ciclo de avisos sin programar.

💬 WhatsApp (Cloud API + wa.me)

Envío de cobros, renovaciones, bienvenida de póliza y seguimientos. wa.me para envío directo; Cloud API (vía Make) para automatización masiva.
Config: token de WhatsApp Cloud API o número para wa.me.
Valor: el canal preferido del cliente, con plantillas y trazabilidad.

✉️ Outlook / Gmail (IMAP/POP3 + OAuth)

Bandeja integrada en la plataforma. Asocia correos a cliente, póliza, gestión o reclamo; redacta con asunto patrón para la aseguradora. Soporta dominios propios.
Config: OAuth (Microsoft 365 / Google) o IMAP/POP3 (servidor, puerto, usuario).
Valor: trazabilidad total de la comunicación en el expediente del cliente.

📗 Green API · Telegram · SMS

Canales alternativos de mensajería para cobros y avisos.
Config: API key del proveedor.

📊 Marketing y contenido

📣 Metricool · Meta (FB/IG) · LinkedIn · TikTok · YouTube

Publicación y medición de contenido del calendario de Marketing en redes; programación y estadísticas.
Config: OAuth de cada red o token de Metricool.
Valor: del calendario Orbit a la publicación real, con métricas de vuelta.

🎨 Canva · Gamma · HeyGen

Generación de piezas gráficas, presentaciones y videos con avatar para capacitación y campañas.
Config: API key / cuenta conectada.

📧 Mailchimp

Campañas de correo masivo segmentadas (renovaciones, novedades).
Config: API key de Mailchimp.

🗂️ Datos y productividad

📁 Google Drive / Sheets / Calendar / Docs

Almacenamiento documental por cliente, exportación a hojas, agenda y documentos.
Config: OAuth de Google Workspace.
Valor: expediente documental real sin migrar archivos manualmente.

🤖 IA — Gemini / ChatGPT / Claude / Endpoint propio

Potencia importadores (extracción/mapeo), comparativo de pólizas, análisis crítico, generación de cursos y copy. Selector sin sesgo en Automatizaciones → Motor de IA.
Config: elige proveedor + modelo + API key. Ninguno preseleccionado.
Valor: la inteligencia real de la plataforma. Ver comparativa-ia.html para elegir.

📓 NotebookLM · Notion · Slack · Trello · Zapier · n8n

Conocimiento, gestión de tareas y automatización adicional según el flujo del cliente.
Config: API key / OAuth según herramienta.

🏛️ Facturación y normativa local

🧾 FEL (SAT Guatemala) · DIAN (Colombia)

Facturación electrónica conforme a la normativa del país configurado. Los patrones de facturación por aseguradora (NIT, concepto) viven en la ficha de cada aseguradora.
Config: credenciales del certificador / DIAN.
Valor: factura con los datos correctos por aseguradora, sin buscar PDFs viejos.

🏦 Open Banking

Conciliación automática del estado bancario con los movimientos de Finanzas.
Config: API del agregador bancario.
