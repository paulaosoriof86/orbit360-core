💡 Recomendación: para extracción de pólizas con garantía de calidad (cotizador / comparativo de A&S), Claude o ChatGPT superan a Gemini Flash. Para volumen y costo (importaciones masivas, OCR de muchos documentos), Gemini es muy rentable validando muestras. En el producto comercial se ofrecen los tres + endpoint propio sin preseleccionar, para que cada cliente elija según su prioridad. A&S: usar Claude o ChatGPT. Para mantener el repo y conectar backend, ChatGPT con Codex es buena opción operativa.
